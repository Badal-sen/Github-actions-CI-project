# Terraform GitHub Actions CI Project

This project demonstrates a CI pipeline using GitHub Actions to validate Terraform code.

---

## 🚀 Tools Used
- Terraform
- GitHub Actions
- AWS (S3)

---

## ⚙️ What it does
- Runs on every push
- Terraform init
- Terraform validate
